# SAFER Security Tool

**Defensive & Legal Network Security Tool**

## Features
- ✅ Full network scan (firewall, ports, DNS, interfaces)
- ✅ Security score & risk level
- ✅ Restore DNS backup
- ✅ View previous reports
- ✅ Check / Change public IP
- ✅ Save custom-named reports
- ✅ Optional silent mode for automation

## Installation
# Clone the repo
git clone https://github.com/Mazen7771/SAFER-Security-Tool.git

# Enter the folder
cd SAFER-Security-Tool

# Make the main script executable
chmod +x safer.sh

##Usage
sudo ./safer.sh
