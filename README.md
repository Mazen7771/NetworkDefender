# SAFER Security Tool

**Defensive & Legal Network Security Tool**  
Check firewall, DNS, services, ports, and public IP. Generates detailed reports with recommendations.

## Features

- ✅ Full network scan (firewall, ports, DNS, interfaces)
- ✅ Security score & risk level
- ✅ Restore DNS backup
- ✅ View previous reports
- ✅ Check / Change public IP## Usage
- ✅ Save custom-named reports
- ✅ Optional silent mode for automation
